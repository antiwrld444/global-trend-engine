# Отчет о рефакторинге GTOE (Global Trend & Opportunity Engine)

## Выполненные работы

### 1. Организация структуры проекта
Проект приведен к профессиональному виду. Созданы и структурированы следующие директории:
- `src/`: Исходный код системы.
  - `analytics/`: Движки скоринга и поиска аномалий.
  - `collectors/`: Модули сбора данных (RSS, News API).
  - `database/`: Управление БД.
  - `utils/`: Общие утилиты (логирование).
- `tests/`: Директория для юнит-тестов (инициализирована).
- `logs/`: Хранение логов приложения.
- `data/`: Хранение локальной базы данных SQLite.

### 2. Внедрение системы логирования
- Создан модуль `src/utils/logger.py` для централизованной настройки логирования.
- Логирование интегрировано во все ключевые модули: `news_collector`, `scoring_engine`, `anomaly_detector`, `db_manager`, `main`.
- Ошибки (уровень ERROR и выше) записываются в файл `logs/app.log`.
- Информационные сообщения выводятся в консоль в реальном времени.

### 3. Обновление документации (README.md)

### 4. Git и синхронизация
- Произведена очистка репозитория от временных файлов (`__pycache__`).
